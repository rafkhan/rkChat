INSTALLATION INSTRUCTIONS

Compile from source

  • Extract the files from the .zip file
  • Compile it javac rkChatServer.java
  • You could either run java rkChatServer
  • Or put it in a .jar jar cfm rkChatServer.jar manifest.mf *class
  • To execute the jar, instead of double clicking it, you must run java -jar rkChatServer.jar
